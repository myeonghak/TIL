Lin Alg 7 -Dot Product Duality

	1.	일반적인 설명
		: 두 벡터가 있을 때, 각 인덱스에 대응하는 원소별로 곱해준 뒤 총합을 수행해 하나의 스칼라 값으로 만들어 주는 연산.
		
	2.	기하적 설명
		: 두 벡터 v와 w가 있을 때, 벡터 w를 벡터 v 위와 원점을 지나게끔 사영시킨다. 이 때 사영된 벡터 v의 길이와 벡터 w의 길이를 곱해주면, 두 벡터의 내적이 계산된다.
		만약 두 벡터의 방향이 반대라면, 내적의 값은 음수가 된다.
		두 벡터의 방향이 직각일 경우, 벡터 w는 원점에서 사영되고, 따라서 내적의 값은 0이 됨.
		
	3.	내적의 이중성
		: 벡터 v와 w에 대한 내적을 수행할 때, 순서는 관계가 없음. v에 w를 사영하든, w에 v를 사영하든 같은 결과를 낳음.
		
		1) 만일 벡터 v와 w의 길이가 같을 경우, 두 벡터의 사영 결과는 완벽한 대칭을 이루게 됨. 따라서 사영된 벡터 v,w의 길이는 각각 동일함. 
		2) 만약 v가 2v가 된다하면, 대칭성은 깨어짐. 그러나 사영된 벡터 w의 길이는 변하지 않음. 단지 v의 길이가 2배가 되었을 뿐이며, 따라서 내적의 값도 2배가 됨.
		반대로 w에 2v를 사영한다 해도, 해석은 같음. 사영된 벡터의 길이가 2배가 되며 w의 길이는 동일함.
		
	4.	일반적 설명과 기하적 설명의 접점 - 이중성
		
		"왜 projection을 통해 내적을 구한 것이, 벡터 간의 element wise product와 같은가?"
	
		1) 2차원->1차원 선형변환에서, 2개의 기저 벡터 i hat과 j hat은 하나의 number line에 안착하게 됨. (1*2 matrix)
		
		2) 1*2 matrix <-> 2d vectors
		1*2 matrix와 2차원 벡터를 곱하는 연산은 벡터의 내적과 동일함.
		
		3) 원점을 지나는 대각 number line를 추가해 보자. 이 number line의 1에 안착하는 unit vector u hat을 그린다. 
		이 number line에 2차원 벡터들을 사영시킨다. 이렇게 하면 사실상, 2차원 벡터를 숫자로 변환하는 함수를 정의한 셈이 됨. 이러한 함수는 선형성 조건을 만족함. 
		여기서, 2차원 벡터가 1차원 상의 하나의 수로 표현됐다는 것에 주목. 
		
		그러나 여기서 u hat은 여전히 2차원 벡터임. 앞선 함수의 결과로 얻어진 1차원 수와 같은 방식으로 위치해 있을 뿐임.
		어쨌든, 이러한 함수는 1*2 행렬로 정의될 수 있음. 2개의 기저벡터가 안착하는 위치를 각 원소로 갖는 이러한 행렬을 찾기 위해, 
		처음에 정의한 diagonal number line을 살펴봄. 그 선 위에 두 기저 벡터가 안착하는 지점이 존재하기 때문.
		
		이 때, i hat과 u hat은 둘 다 단위 벡터이기 때문에, 대칭성이 성립하고, 
		i hat이 diagonal number line이자 u hat이 존재하는 선 위에 사영된 길이와 u hat이 x축 위에 사영된 길이는 완벽히 동일하게 됨. 
		그런데 이 때 u hat을 x 축에 사영한다는 것은, 단지 u hat의 x축 좌표를 의미함! 동일한 원리로,	j hat을  number line에 사영한 길이는 u hat의 y축 좌표를 의미함.
		
		
		따라서 u hat의 x,y 좌표는 각각 앞서 찾고자 했던  1*2 행렬(사영 변환을 나타내는 함수) 각 원소 값, 
		즉 단위 벡터가 안착하는 지점을 나타내는 scaling factor가 됨. 따라서 이 행렬과 2차원 벡터를 곱해주는 연산은 완벽히 u hat과의 내적과 동일하게 됨. 

		바로 이것이 단위 벡터와 내적하는 것이 단위 벡터의 span에 대상 벡터를 사영하는 것과 동일하게 해석될 수 있는 이유임. 

		단위벡터가 아닌 경우에도(non-unit vector) 이는 일반화될 수 있음. 단순히 사영한 뒤 scaling해주는 것과 동일함. 

		결론: 1*2 행렬을 2차원 벡터와 곱하는 연산은 2차원 벡터와 2차원 벡터를 내적한 것과 같다!
