[트리]

1. 정의: 정점(node)과 간선(edge)을 이용하여 데이터의 배치 형태를 추상화한 자료구조

2. 나무의 뿌리(root)가 있고, 이파리(leaf)가 달린 형태. 나무를 거꾸로 매달린 형태로 표현.

3. 구성
	root 노드: 맨 위의 노드
	leaf 노드: 마지막에 달린 자식이 없는 노드
	부모 노드: 연결된 노드 구조에서, 더 뿌리 쪽에 가까운 노드.
	자식 노드: 연결된 노드 구조에서, 더 잎 쪽에 가까운 노드.
	
	조상(ancestor): 부모의 부모 (의 부모의 ...)
	후손(descendant): 자식의 자식 (의 자식의 ...)
	
	노드의 수준(level): root 노드로부터 해당 노드에 이르기까지 거치는 간선의 수.
	
	높이(height)/깊이(depth) : 최대 수준 + 1
	
	부분트리(subtree): 어떤 트리에서 노드 1개로부터 후손들을 모두 떼어낸 것.
	
	노드의 차수(degree): leaf를 향해 연결된 node의 수.
	

4. 이진트리(binary tree)
	1) 정의: 모든 노드의 차수가 2 이하인 트리
	2) 재귀적 정의 가능:
		루트노드 + 왼쪽 서브트리 + 오른쪽 서브트리
		(단, 이 때 왼쪽과 오른쪽 서브트리 또한 이진트리)
		
		+ 빈 트리도 이진트리이다. (재귀를 중단할 수 있는 terminal 조건)
		
		
5. 포화 이진 트리 (Full Binary Tree)
	1) 정의: 모든 레벨에서 노드들이 모두 채워져 있는 이진트리
	2) 높이가 k이고, 노드의 개수가 2^k -1 인 이진트리
	
6. 완전 이진 트리 (Complete Binary Tree)
	1) 정의: 높이 k인 완전 이진트리는 다음과 같이 정의됨
		-> 레벨 k-2까지는 모든 노드가 2개의 자식을 가진 포화 이진트리
		-> 레벨 k-1에서는 왼쪽부터 노드가 순차적으로 채워져 있는 이진 트리
	2) 마지막 leaf는 왼쪽부터 차근차근 채워나가게 됨.
