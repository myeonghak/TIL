1장. 숫자

컴퓨터는 정부롤 비트로 표현함. 2진법 뿐만 아니라 8진법, 16진법등의 2의 배수 진법도 사용함.

1.1 정수
int는 불변형(immutable)임. 불변형 객체는 변수와 객체 참조간에 차이가 없음.
정수는 컴퓨터 메모리에 의해 제한됨. 이는 C/자바 내장 컴파일러에 따라 다름.
파이썬 정수 크기는 최소 32비트(4바이트)임. 

정수 표현에 필요한 바이트 수를 확인하기 위해서는, 
(int).bit_length() 메서드를 사용하면 됨.

int("문자열", 밑)
-> 다른 진법의 문자열을 정수로 변환하는 메서드

1.2 부동소수점
파이썬의 부동소수점은 float으로 나타내며 불변형임.
단정도 방식(single precision)에서 32비트 부동소수점을 나타낼 때 1비트는 부호(1은 양수, 0은 음수)를 나타내며, 23비트는 유효숫자 자릿수(significant digits)를, 8비트는 지수(exponent)임.

	1.2.1 부동소수점끼리 비교하기
	동등성 테스트를 실행할 경우, 같은 수를 낳아야함에도 그렇지 않은 경우가 발생함.
	이 경우 사전에 정의된 정밀도의 범위 내에서 수행되어야 함.
	unittest 모듈의 assertAlmostEqual() 메서드 같은 접근법을 사용하는 방법등이 존재함.
	
	1.2.2 정수와 부동소수점 메서드
	파이썬에서 나누기 연산자 /는 항상 부동소수점을 반환함.
	//(floor, truncation)을 사용하면 정수를 반환할 수 있음.
	% (module, remainder)연산자는 나머지를 구함.
	divmod(x,y) 메서드는 x를 y로 나눌 때, 몫과 나머지를 반환함.
	
	round(x,n) 메서드는 n이 음수인 경우, x를 n만큼 반올림한 값을 반환함.
		n이 양수인 경우, x를 소수점 이하 n자리로 반올림.
		
	
	as_integer_ratio() 메서드는 부동소수점을 분수로 표현함.

1.3 복소수 (complex number)
파이썬에서 복소수는 z=3+4j와 같이 생긴 부동소수점 한쌍을 갖는 불변형임.
z.real, z.imag, z.conjugate()와 같은 메서드로 실수부/허수부/켤레복소수를 구할 수 있음.

복소수를 사용하기 위해서는 cmath 모듈을 임포트 해야함.
cmath 모듈은 math 모듈에 있는 대부분의 삼각함수, 로그함수의 복소수 버전 및 복소수 전형 함수를 제공함.

1.4 fraction 모듈
분수를 다루기 위한 내장 모듈.

1.5 decimal 모듈
정확한 10진법 부동소수점 숫자가 필요한 경우, decimal.Decimal을 사용할 수 있음.
Decimal() 메서드는 정수/문자열을 인수로 취함.



1.6 2진수, 8진수, 16진수
bin(i) 메서드는 정수 i의 2진수 문자열을 반환함.
oct(i) 메서드는 정수 i의 8진수 문자열을 반환함.
hex(i) 메서드는 정수 i의 16진수 문자열을 반환함.
